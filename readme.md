# Form Builder

A lightweight, dependency-free form builder that lets you visually create HTML forms and export their code.

## Features

- Add labels, text inputs, and buttons via the sidebar panel
- Live preview of the form on the left panel
- Delete individual elements from the preview
- Generate and copy the resulting HTML code
- Fully responsive layout with flexbox
- No external dependencies (vanilla JS)

## Usage

Open `index.html` in a browser. Use the buttons on the right panel to add elements:

1. **Label** - adds a `<label>` with custom text
2. **Zone de texte** - adds an `<input>` with a custom id
3. **Bouton** - adds a `<button>` with custom text
4. **Code Formulaire** - generates the HTML code and lets you copy it
